

# EE471 Mini Project 2 – Voice Activated Calculator

This project is a voice-activated calculator developed for EE471 Modern Software Development Practices and Technologies.

## Features
- Button-based calculator interface
- Voice output for button presses and results
- Voice input through microphone
- Supports basic arithmetic operations:
  - Addition
  - Subtraction
  - Multiplication
  - Division

## Voice Commands
Examples:
- Five plus two is
- Seven minus three is
- Three times two is
- Nine over three is

## Technologies Used
- Python
- Tkinter
- SpeechRecognition
- pywin32
